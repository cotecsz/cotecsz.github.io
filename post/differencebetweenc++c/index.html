C++与C的异同

2020-05-26
cpp
约 871 字 预计阅读 2 分钟

文章目录

  • 全局变量检测增强

  • 函数定义要求更为严格

    • 参数类型检测：C 语言中，对于参数类型检测不严格，而 C++ 严格检测参数类型
    • 是否返回值类型：在 C 语言中，定义函数有返回值，在函数结束未返回，程序正常运行；在 C++ 中，若声明有返回值，不返回返回值则编译不通过。
    • 函数调用参数检测

  • 类型转换检测增强

    • C 语言中，void* 可以表示任何类型指针
    • C++ 中，必须强制转换

2
3

void getSpace(){
  	char* p = malloc(sizeof(32));	// malloc 返回 void *
}

void getSpace(){
  	char* p = (char*)malloc(sizeof(32));	// malloc 返回 void *
}

 

  • Struct 增强：C 语言中，声明 struct 对象时，必须使用 struct 关键字，在 C++ 中不需要关键字

  • bool 类型增强：C 语言中没有 bool 类型，C++ 中 包含 bool 类型

  • 三目运算符增强：C 语言中，三目运算符返回的是 值 ，而 C++ 中的三目运算符返回的是 变量 。

1

(a > b ? a : b) = 100;	// C 语言中， 相当于20 = 100 C 语言中返回的是值

1
2

(a > b ? a : b) = 100;	// C++ 中， b = 100
												// C++ 返回的是变量

 

  • const 增强
    • C 语言中，局部变量的 const 是伪常量，编译器会给变量分配内存，可以通过指针修改，故不可初始化数组；全局变量的 const 修饰的变量在全局区域中，不可改变。而 C++ 中，局部变量的 const 是常量，在符号表中存储。
    • C 语言中，const 默认是外部链接；C++ 中， const 默认是内部链接，可以使用 extern 提高作用域

const int a = 10;
void changeConst(){
  	const int b = 20;
  	int *pB = (int*)&b;		// 临时开辟内存空间,相当于
  												// int tmp = b;
  												// int *p = (int*) &tmp;
  	*p = 200;
  	cout << "*p = " << *p << endl;	// *p = 200
  	cout << "b = " << b << endl;		// b = 20
  
  	int arr[b];		// 正确， b 是常量
}

 

  • const 默认不会分配内存，在符号表中存储，但分配内存的三种情况
    • 取地址时，分配临时内存
    • extern 编译器给 const 变量分配内存
    • 用变量初始化 const 变量
    • 自定义数据类型使用 const 修饰时，也会分配内存

注意：只要分配内存，则可以通过指针改变 const 修饰的对象。

  • 尽量使用 const 替换 #define
