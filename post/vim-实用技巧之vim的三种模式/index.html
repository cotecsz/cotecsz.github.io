普通模式

技巧 7 停顿时转移至普通模式

  • 在完成编辑时，按 ESC 调整至 普通模式。

 

技巧 8 把撤销单元切成块

  • 修改单元：从进入插入模式开始，知道返回普通模式为止，在此期间输入/删除的任何内容被当成一次修改。

  • 使 可撤销单元 对应一次思考过程，在一次思考过程后进入插入模式，编辑，进入普通模式，循环往复形成不同的可撤销单元。

 

技巧 9 构造修改可重复

  • 充分利用 . 命令，在进行一次操作时，尝试构造修改，使得 . 命令可重复。

2
3

例：删除整个单词(无论光标在何处)
daw		// delete around word
.			// repeat daw

  • Ctrl + a ：对数字执行加法

  • Ctrl + x ：对数字执行减法

技巧12 双剑合壁，天下无敌

操作符 + 动作命令 = 操作

命令 用途
c 修改
d 删除
y 复制到寄存器
g~ 反转大小写
gu 转换为小写
gU 转换为大写
> 增加缩进
< 减小缩进
= 自动缩进
! 使用外部程序过滤{motion}所跨越的行

 

插入模式

技巧 13 在插入模式中可及时更正错误

按键操作 用途
Ctrl + h 删除前一个字符
Ctrl + w 删除前一个单词
Ctrl + u 删至行首

 

技巧 14 返回普通模式

按键操作 用途
ESC 切换到普通模式
Ctrl + [ 切换到普通模式
Ctrl + o 切换到插入-普通模式

 

插入-普通模式

  • 在插入模式下，可以按 Ctrl + o 进入插入 - 普通模式，使用普通模式的命令。

 

技巧 15 不离开插入模式，粘贴寄存器中的文本

  • 使用 Ctrl + r + {register} 可在插入模式粘贴 某个寄存器的内容。

 

技巧 16 随时随地做运算

  • 在插入模式中， Ctrl + r + = [运算]

 

技巧 17 用字符编码插入非常用字符

按键操作 用途
Ctrl + v + {123} 以十进制字符编码插入字符
Ctrl + v + u{1234} 以十六进制字符编码插入字符
Ctrl + {nondigit} 按原意插入非数字字符
Ctrl + k + {char1}{cahr2} 插入以二合字母{char1}{char2}表示的字符

技巧 19 用替换模式替换已有文本

  • R 进入 替换模式，以文件中所保存的字符来替换字符。

  • gR 进入替换模式，以屏幕实际显示的宽度来替换字符。（推荐）

  • r{char} 替换单个字符

  • gr{char} 替换单个字符

 

可视模式

技巧 20 深入理解可视模式

  • 可视模式与普通模式相同，可以使用 h , j , k , l, f{char} , ;, , , n 和 N 进行选择。

 

技巧 21 选择高亮选区

三种可视模式

命令 用途
v 激活面向字符的可视模式
V 激活面向行的可视模式
Ctrl + v 激活面向列块的可视模式
gv 重选上次的高亮选区

 

  • 在可视模式之间，可相互切换
  • o 切换选区的活动端

 

技巧 22 重复执行面向行的可视命令

  • 当对行进行操作时，尽量使用 . 进行重复

 

技巧 23 只要可能，最好用操作符命令，而不是可视命令

  • 可视模式在使用 . 时，含产生意想不到的情况，尽量在普通模式使用操作符命令

 

技巧 24 面向列块的可视模式编辑表格数据

  • Ctrl + v 进入列块选择
  • gv 重选上次的高亮选区

 

技巧 25 修改列文本

  • Ctrl + v 进入列块选择可视模式
  • 修改时，插入操作之影响顶行，在退出插入模式后，其他行也会被修改

技巧 26 在长短不一的高亮块后添加文本

  • Ctrl + v 进入列块选择可视模式
  • A 可以在每行的结尾添加内容

 

在列块可视模式中，a 和 i 为一个文本对象的组成部分，可以使用 I 和 A 分别在列块开头和结尾处插入。

 

命令行模式

技巧 27 认识 Vim 的命令行模式

命令 用途
:[range]delete [x] 删除指定范围的行至寄存器 x
:[range]yank [x] 复制指定范围的行至寄存器 x
:[line]put [x] 在指定行后粘贴寄存器 x 中的内容
:[range]copy {address} 将指定范围的行拷贝至{address}行之下
:[range]move {address} 将指定范围的行移动至{address}行之下
:[range]join 连接指定范围的行
:[range]normal {commands} 对指定范围的每一行执行普通模式命令
:[range]substitute/{pattern}/{string}/[flags] 把指定范围出现{pattern}替换为{string}
:[range]global/{pattern}/[cmd] 对指定范围匹配{pattern}执行Ex命令{cmd}

 

技巧 28 在一行或多个连续行上执行命令

  • 用行号作为地址，:3p
  • 用地址指定一个范围，:3, 5p ，. , 5
  • 用高亮选区指定范围，'<'>
  • 用模式指定范围
  • 用偏移对地址进行修正，{address}+n
符号 地址
1 文件的第一行
$ 文件的最后一行
0 虚拟行，位于文件第一行上方
. 光标所在行
‘m 包含位置标记 m 的行
‘< 高亮选区的起始行
‘> 高亮选区的结束行
% 整个文件

 

技巧 29 使用 :t 和 :m 命令复制和移动行

命令 用途
:6t. 把第6行复制到当前行的下方
:t6 把当前行复制到第6行下方
:t. 为当前行创建一个副本
:t$ 把当前行复制到文件结尾
:‘<,‘>t0 把高亮选区复制到文件开头

技巧 35 运行 shell 命令

  • 在命令前加一个 ! 即执行 shell 命令，输出后返回 vim
  • 需要执行多个命令时，使用 :shell 将 vim 置于后台，exit 返回 vim
  • 使用 Ctrl + z 将当前 vim 置于后台，fg 返回 vim
  • read !{cmd} 将命令输出至当前缓冲区
  • write !{cmd} 将缓冲区的内容作为标准输入
命令 用途
:shell 启动一个 shell （输入exit 返回 Vim）
:!{cmd} 在 shell 中执行 {cmd}
:read !{cmd} 在 shell 中执行 {cmd}，并将其标准输出插入光标下方
:[range]write !{cmd} 在 shell 中执行 {cmd} ，以 [range] 作为其标准输入
:[range]!{filter} 以外部程序 {filter} 过滤指定的 [range]

 

技巧 36 批处理运行 Ex 命令

  • 将命令保存至 cmd.vim
  • 在需要处理的文件中，执行 Ex 命令 ：:source cmd.vim
